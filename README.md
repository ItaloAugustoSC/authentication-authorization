# Sistema de Autenticação e Autorização Completo em Java

Essa implementação usará **Spring Boot** + **Spring Security** + **OAuth2** + **JWT**, permitindo autenticação com provedores externos (**Google, Facebook**), além de autenticação via login e senha.

## Tecnologias e Ferramentas
- **Spring Boot** (para desenvolvimento rápido)
- **Spring Security** (para autenticação e autorização)
- **JWT (JSON Web Token)** (para autenticação stateless)
- **OAuth2** (para login social)
- **Banco de Dados (PostgreSQL)** (para armazenar usuários e permissões)
- **Docker** (para facilitar a configuração do ambiente)
